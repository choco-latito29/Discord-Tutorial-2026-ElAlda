# 🔗 External Resources & Useful Links

A curated list of all the tools, documentation, and communities mentioned throughout the **ELALDA 2026** tutorial series.

## 🛠️ Official Tools & Portals

- **[Discord Developer Portal](https://discord.com/developers/applications)**: Where you create your bot, get your Token, and manage Gateway Intents.
- **[Discord.js Official Documentation](https://discord.js.org/)**: The ultimate guide for the library used in this project (v14).
- **[Node.js Downloads](https://nodejs.org/)**: Recommended version for this bot: **Node.js 22.x**.

## 📦 Key Libraries Used

- **[discord.js](https://www.npmjs.com/package/discord.js)**: The core library to interact with the Discord API.
- **[@napi-rs/canvas](https://www.npmjs.com/package/@napi-rs/canvas)**: The high-performance Rust-based library for image manipulation (Love/Ship commands).
- **[dotenv](https://www.npmjs.com/package/dotenv)**: Essential for managing environment variables and protecting your Token.
- **[colors](https://www.npmjs.com/package/colors)**: Used for pretty and organized terminal logs.

## 🚀 Hosting & Deployment
